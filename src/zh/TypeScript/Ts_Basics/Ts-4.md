### 给变量添加基本类型
:::info 说明
- `TypeScript` 里面的基本类型有 `string` `number` `boolean`

- 在上一章节中我们使用的是隐式推断，这一章节我们来学习显式推断
:::

#### 显式推断
```ts
// 语法 let 变量名: 类型 = ''
let prductName: string = ''

let pric: number = 200

let isShow: boolean = true
```
