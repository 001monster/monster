### 给数组变量添加类型
#### 隐式写法
- ts 隐式推断出的类型是 let nums: number[]
- 这样意思就是定义完以后数组里面的元素就必须使用 number, 后面就不可以修改了
```ts
let nums = [1, 2, 3]
```

#### 显式写法
```ts
let nums: number[] = [1, 2, 3]
```

#### 使用构造函数的写法（泛型）
```ts
let nums: Array<number> = [1, 2, 3]
```

#### 对象数组合用
```ts
interface Product {
    title: string,
    price: number
}

let products: Product[] = [
    {
        title: '纯棉秋裤',
        price: 200
    },
    {
        titlle: '纯棉秋衣',
        price: 199
    }
]
```
